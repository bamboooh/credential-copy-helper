# 資格情報コピー支援ツール

## 概要

各種アカウント情報や資格情報を一覧で管理し、ワンクリックでクリップボードへコピーするためのツール。

ブラウザでHTMLファイルを開くだけで動作するため、インストール不要・オフライン環境でも利用可能。

---

## 特徴

・ダブルクリックで起動可能（Webサーバ不要）
・ボタンを押すだけでコピー可能
・表示文字列とコピー内容を分離可能（伏字対応）
・データ追加はファイル1箇所のみ

---

## ファイル構成

```
tool/
├─ index.html        // 画面
├─ copy_btn.css      // 見た目
├─ app.js            // 処理
└─ data.js           // データ（ここだけ編集すればOK）
```

---

## 使い方

1. `index.html` をダブルクリック
2. ブラウザで画面が表示される
3. コピーしたい項目のボタンをクリック
4. クリップボードにコピーされる

---

## データ追加方法（重要）

編集対象は **data.js のみ**

### 基本フォーマット

```javascript
{
    category: "カテゴリ名",
    title: "項目名",
    label: "画面に表示する文字",
    value: "実際にコピーされる文字列"
}
```

---

### 追加手順

1. `data.js` を開く
2. 配列 `COPY_ITEMS` の中に追記する
3. カンマ `,` を忘れずに追加する

---

### 例（1件追加）

```javascript
const COPY_ITEMS = [
    {
        category: "既存カテゴリ",
        title: "既存項目",
        label: "表示文字",
        value: "コピー内容"
    },

    // ↓ここに追加
    {
        category: "新しいカテゴリ",
        title: "サンプルID",
        label: "ABC123",
        value: "ABC123"
    }
];
```

---

## 各項目の意味

| 項目       | 説明           |
| -------- | ------------ |
| category | カテゴリ（グループ名）  |
| title    | 項目名（左側に表示）   |
| label    | ボタンに表示される文字  |
| value    | コピーされる実際の文字列 |

## よくある使い方

### パスワードを伏字表示にする

```javascript
{
    category: "ログイン情報",
    title: "パスワード",
    label: "********",
    value: "ActualPassword123"
}
```

---

### 同じカテゴリに追加する

```javascript
{
    category: "Dコネ/TSC認証",
    title: "新しい項目",
    label: "XXXX",
    value: "XXXX"
}
```

※ category名が同じなら同じグループに表示される

## NG例（壊れる原因）

### カンマ忘れ

```javascript
{
    title: "NG例"
}   // ←ここにカンマがないとエラー
{
    title: "次の要素"
}
```

---

### クォート抜け

```javascript
title: サンプル   // ←ダメ
title: "サンプル" // ←正しい
```

---

### プロパティ名ミス

```javascript
catgory: "NG"   // ←ダメ
category: "OK"  // ←正しい
```

## トラブルシュート

### 画面が真っ白になる

→ data.js の記述ミスの可能性が高い
→ カンマ・クォートを確認

---

### コピーできない

→ 一部ブラウザ制限の可能性
→ Edge / Chrome で開く

---

### 表示が崩れる

→ CSSファイルの破損
→ copy_btn.css を確認
